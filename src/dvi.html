3DViewer_v1.0

О программе

Программа реализована на графических библиотеках QT и Opengl. С помощью нее пользователь может:

1. Просматривать объекты, загружая файлы c расширением .obj.

2. Двигать, вращать, маштабировать загруженный пользователем обьект.

3. Выбирать тип линий, вершин.

4. Выбирать цвет фона, линий, вершин.

5. Выбирать размер вершин и линий.

6. Выбирать вид проекции.

7. Сохранять получившийся результат в форматах .jpeg, .bmp, .gif.

Как начать

Чтобы начать пользоваться программой, ее необходимо установить с помощью команды make install. Эта команда установит программу на рабочий стол в папку 3DViewer_v1.0.

Как Пользоваться

• Чтобы загрузить объект, нажмите "Открыть файл" и укажите путь до него

• Чтобы сдвинуть или повернуть обьект, введите в строку значение и нажмите на кнопку с наименованием желаемой оси. Положительное значение - поворот по часовой, сдвиг вправо/вверх/на себя в зависимости от выбранной оси. Отрицательное значение - поворот против часовой, сдвиг влево/вниз/от себя в зависимости от выбранной оси. Если вы хотите вернуть обьект в исходное положение, нажмите кнопку "СБРОС".

• Чтобы применить маштаб введите в строку значение: для уменьшения число от 0 до 1, для увеличения число больше 1 и нажмите кнопку "Применить маштаб".

• Чтобы использовать настройки(цвет, размер и др.) выберите желаемые и нажмите кнопку "Применить". Данная программа сохраняет значения настроек послее ее закрытия. Чтобы сбросить настройки, нажмите кнопку "Сброс настроек".

• Чтобы сохранить получившийся результат в формате изображения, нажмите на кнопку "Save image" и программа предложит вам указать имя файла и путь для сохранения.

• Чтобы сохранить получившийся результат в формате анимации, нажмите на кнопку "Save gif", начнется 5-ти секундная запись, во время которой, вы можете совершать возможные действия с объектом. По истечению времени, программа предложит вам выбрать имя файла и путь для сохранения.

Удалить

Удалить пользовательскую программу вы можете использовав команду make uninstall.
